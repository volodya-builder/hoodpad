Верификация контрактов hood — мейннет Robinhood Chain

Сеть: Robinhood Chain Mainnet · Эксплорер: robinhoodchain.blockscout.com

Как верифицировать (для каждого контракта одинаково)

1. Скачай файл verification-input-mainnet.json (это исходники всех контрактов одним файлом).
2. У каждого контракта ниже нажми «Открыть форму верификации».
3. В форме выбери способ: Solidity (Standard JSON input).
4. Compiler version: v0.8.28+commit.7893614a
5. Загрузи скачанный verification-input-mainnet.json.
6. Contract name — впиши точное имя из карточки (например LaunchpadFactory).
7. Аргументы конструктора Blockscout определит сам. Нажми Verify.
